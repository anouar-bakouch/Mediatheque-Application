# Mediatheque Application 

Frontend : JavaSwing Graphic interfaces
Backend : Java JEE Architecture
Database : XML

### Prerequisites
- java JDK 19
- jdom 2.0.6

# Getting Started
- Clone the project
- Import the project in your IDE
- Run the project
- Enjoy
